# AI-Based Clinical Assistance for Ayurvedic Drug Dosage

## 📌 Description
This project provides personalized Ayurvedic drug dosage recommendations based on user inputs such as age, weight, and symptoms.

## 🚀 Features
- Predicts dosage level (Low, Medium, High)
- User input-based recommendation system
- Simple AI/ML or rule-based logic

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn

## ⚠️ Disclaimer
This project is for educational purposes only and not intended for real medical use.

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python app.py
